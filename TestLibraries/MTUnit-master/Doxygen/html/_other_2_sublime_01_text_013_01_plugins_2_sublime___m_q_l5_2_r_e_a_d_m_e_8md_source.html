7 Installation
8 -------------
9 Make sure that you have MetaTrader5 installed.
10 
11 #### Using Package Control
12 If you already have [Package Control](http://wbond.net/sublime_packages/package_control/) installed in Sublime Text:
13 
14 * Select "Install Package" from the Command Palette: <kbd>Ctrl+Shift+P</kbd> on Windows and Linux or <kbd>⇧⌘P</kbd> on OS X)
15 * Search for "**mql5**" and press <kbd>enter</kbd>.
16 
17 #### Manual Installation
18 Go to `Preferences -> Browse Packages -> User`, and then either download and unzip this plugin into that directory, or:
19 
20 ``` bash
21 git clone https://github.com/rodrigopandini/sublime_mql5.git "sublime_mql5"
22 ```
23 
24 Configuration
25 -------------
26 Configure the `buildMQL5.bat` file to point to the correct location of `metaeditor64.exe` program and include path.
27 ```
28 ...
29 set metaeditor="C:\Program Files\MetaTrader 5\metaeditor64.exe"
30 set include_path="C:\Users\User\AppData\Roaming\MetaQuotes\Terminal\XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX\MQL5"
31 ...
32 ```

34 Use
35 ---
36 **Compile or Syntax Check**
37 <kbd>Ctrl+B</kbd> or <kbd>F7</kbd> to compile and build your project.
38 <kbd>Ctrl+Shift+B</kbd> and select "Syntax Check" to check the syntax in your project without compilation.
39 This will run the MetaEditor in command-line mode and display the results in the Sublime Text console.
40 When result data is captured, you can navigate to results in your project’s files with <kbd>F4</kbd> and <kbd>Shift+F4</kbd>. If available, the captured error message will be displayed in the status bar.
41 
42 **Syntax Highlight**
43 Sublime Text will highlight your `.mq5` and `.mqh` files in a convenient way. Since MQL5 and C/C++ are "sisters" languages, the highlight is very similar to C/C++.
44 
45 **Auto-Complete and Snippets**
46 Start typing your code and Sublime Text will display the auto-complete options to accelerate your production. There is a bunch of common and prepocessor snippets, but you can create your own.
